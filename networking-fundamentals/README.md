# Networking Fundamentals

## Overview
This section documents foundational networking concepts and applied learning supporting infrastructure design, segmentation, and secure system communication.

## Focus Areas
- IP addressing and subnetting
- Routing and switching concepts
- Network segmentation and isolation
- Traffic flow and pathing
- Foundational security controls

## Real Context
Networking concepts are applied through hands-on lab environments, including a physically isolated home network and cloud-based infrastructure in Azure. These experiences support understanding of how devices communicate across systems and how networks are structured and secured.

## Portfolio Alignment
This area supports:
- Home network lab (segmentation and infrastructure design)
- Azure cloud networking (VNet, NSG, connectivity)
- OT/energy infrastructure concepts (segmentation and system boundaries)
- Security-focused architecture design
